Let`s talk about html tags for absolute beginners


Parts of HTML code

HTML code consist of two basic parts: head and body

Head

Information in head part is almost invisible for user. It usualy contains information about document formating.
Only part visible for an user is "title" tag

head usuallylinks to a CSS file with styles of the webpage.

Example of link:

< link rel="stylesheet" type="text/css" href="StyleSheet1.css" />

Body

Actual content of an webpage. Most of magic happens here
